# lmaa.space Status

Uptime monitoring and status page for [lmaa.space](https://lmaa.space), live at **[status.lmaa.space](https://status.lmaa.space)**.

Endpoints are probed by [Upptime](https://upptime.js.org) (GitHub Actions write the results to `history/`) and the page is rendered by [Velvet](https://github.com/phranck/velvet). Incidents and maintenance windows are tracked as GitHub Issues.
